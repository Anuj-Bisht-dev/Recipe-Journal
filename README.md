# Recipe Journal

A website to solve a real-life problem of a man. who carries multiple of recipe books by building a website based on his journals recipes.

### Getting started

```bash
 git clone https://github.com/Anuj-Bisht-dev/Recipe-Journal.git
```

**To run the file first use** :

**step 1:** ` npm install`<br>

**step 2:** `npm run dev`<br>

**step 3:** `open project in live server`
